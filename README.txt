カルテ解析ツール - セットアップ手順
=====================================

【必要なもの】
・Windows PC
・NVIDIAのGPU（推奨：VRAM 8GB以上）
・インターネット接続（初回セットアップのみ）

=====================================
【手順1】Pythonのインストール
=====================================
1. https://www.python.org/downloads/ を開く
2. 「Download Python」をクリック
3. インストール時に「Add Python to PATH」に必ずチェックを入れる
4. インストール完了

=====================================
【手順2】Ollamaのインストール
=====================================
1. https://ollama.com/download を開く
2. 「Download for Windows」をクリック
3. インストール完了

=====================================
【手順3】必要なライブラリのインストール
=====================================
コマンドプロンプトを開いて以下を実行：

python -m pip install streamlit docling reportlab ollama

【手順4】AIモデルのダウンロード
=====================================
コマンドプロンプトで以下を実行：

ollama pull gpt-oss:20b

※13GBあるのでダウンロードに時間がかかります

=====================================
【手順5】ファイルの配置
=====================================
以下のファイルを同じフォルダに置く：
・app.py
・main.py
・llm_client.py
・起動.bat

=====================================
【手順6】起動
=====================================
「起動.bat」をダブルクリックするとブラウザが開きます

=====================================
【モデルの保存先を変更したい場合】
=====================================
Cドライブの容量が少ない場合：
1. Windowsキー→「環境変数」で検索
2. 「システム環境変数の編集」を開く
3. 「新規」で以下を追加：
   変数名：OLLAMA_MODELS
   値：任意のパス（例：D:\ollama\models）
4. PCを再起動

=====================================
【GPU2枚使う場合】
=====================================
1. Windowsキー→「環境変数」で検索
2. 「システム環境変数の編集」を開く
3. 以下の環境変数を追加：
   CUDA_VISIBLE_DEVICES = 0,1
4. PCを再起動

=====================================
【トラブルシューティング】
=====================================
・「Ollamaに接続できない」→ 起動.batからやり直す
・「モデルが見つからない」→ ollama listで確認、なければollama pull gpt-oss:20bを実行
・「ライブラリが見つからない」→ 手順3を再実行

=====================================
【使い方】
=====================================
1. 起動.batをダブルクリック
2. ブラウザが開いたらPDFをアップロード
3. 「解析開始」をクリック
4. 結果が表示されたらCSV/TXT/PDFでダウンロード可能

=====================================
【注意事項】
=====================================
・患者情報を含むデータはこのPC内のみで処理されます
・外部サーバーには一切送信されません
・PCを再起動するとOllamaは停止します（起動.batで再起動してください）
